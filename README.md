# Proyectos con PIC16F887

## 📋 Introducción

Este repositorio contiene proyectos desarrollados utilizando el microcontrolador **PIC16F887**. El PIC16F887 es un microcontrolador de 8 bits de la familia PIC de Microchip, ideal para aplicaciones de sistemas embebidos debido a sus características:

- 40 pines con múltiples funciones I/O
- Amplio rango de voltaje de operación (2.0V - 5.5V)

## 🎯 Ejercicios Desarrollados

### Ejercicio 1: Secuencia de LEDs
**Descripción:** Implementación de una secuencia programable de 8 LEDs conectados al puerto C del PIC16F887. El proyecto incluye switches para controlar diferentes patrones de secuencia luminosa.

![Ejercicio 1 - Secuencia LEDs](https://github.com/jhon23-2/Projects_embed_system-Public_/blob/main/exercise1.X/EXERSICE_ONE.gif)

### Ejercicio 2: Sequencia de codigo de gray 
**Descripción:** Implementación de una secuencia programable de 4 LEDs conectados al puerto D del PIC16F887 que muestra una sequencia de codigo de gray por medio de los leds
![Ejercicio 2 - Gray Code LEDs](https://github.com/jhon23-2/Projects_embed_system-Public_/blob/main/gray_code.X/GRAY_CODE.gif)


### Ejercicio 3: Display 7 segmentos (4 displays)
**Descripción:** Implementación de un contador de 0000 hasta 9999 con displays 7 segmentos con el microcontrolador PIC16F887 
![Ejercicio 3 - Gray Code LEDs](https://github.com/jhon23-2/Projects_embed_system-Public_/blob/main/exercie3.X/EXERCISE_3.gif)

---

### Ejercicio 4: Display 7 Segmentos Cátodo Común
**Descripción:** Control de display de 7 segmentos de cátodo común utilizando el PIC16F887. El sistema incluye switches para seleccionar diferentes dígitos y compuertas lógicas para el manejo del display.

![Ejercicio 4 - Display 7 Segmentos](https://github.com/jhon23-2/Projects_embed_system-Public_/blob/main/exercie4.X/EXERCISE_FOUR.gif)

---

### Ejercicio 5: Manipulación de LCD 16x2
**Descripción:** Interfaz con pantalla LCD de 16x2 caracteres (LM016L) conectada al PIC16F887. El proyecto demuestra la escritura de texto y control completo del display LCD.

![Ejercicio 5 - LCD 16x2](https://github.com/jhon23-2/Projects_embed_system-Public_/blob/main/exercie-5.X/EXERCISE_FIVE.gif)
